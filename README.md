## Tech Stack

**Client:** ReactJS, TailwindCSS

## Run Locally

Clone the project

```bash
  git clone hhttps://github.com/jgsamaaa/gym-website.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
